# Rock Paper Scissors Game

A classic Rock Paper Scissors browser game built with vanilla HTML, CSS, and JavaScript.

[![Live Site](https://img.shields.io/badge/Live%20Site-000?style=for-the-badge&logo=github&logoColor=white)](https://jainishjain11.github.io/Rock-Paper-Scissor/)

## 🎮 Demo
![Rock Paper Scissors Demo](./site%20demo/final_63580dce49fc0921e9547f89_194957.gif)

## 📱 Features
- Play with Rock ✊, Paper 🤚, or Scissors ✌️
- Keyboard shortcuts: **R**, **P**, **S**
- Live score tracking
- High score persistence
- Responsive design
- Sound effects

## 🕹️ How to Play
- **Rock** beats Scissors
- **Scissors** beats Paper  
- **Paper** beats Rock
- Same choices = tie (play again!)

## 📂 Run Locally
```bash
# Clone the repo
git clone https://github.com/jainishjain11/Rock-Paper-Scissor.git

# Navigate to project
cd Rock-Paper-Scissor

# Start local server
python3 -m http.server 8000

# Open in browser
http://localhost:8000
```
